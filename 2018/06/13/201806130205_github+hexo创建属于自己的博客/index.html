github+hexo创建属于自己的博客

发表于 2018-06-13 | 分类于 hexo

工作生活中看的东西，不管是技术还是文章很杂乱,总结性的东西没有记录，到自己用时却沿着之前的路去baidu，google,记录下来期望节约时间.

之前一直羡慕别人的博客,现在记录下自己搭建hexo+github博客,希望能帮助到大家.
其中学习参考以及查询许多人经验，已记录在文章结尾。

2: 本地安装git

3: 本地安装node

5: 启动本地hexo查看效果

6：连接本地hexo与github page

7：发布hexo并访问地址 https://yourName.github.io 默认信息 hello word
